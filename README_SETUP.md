# CineVerse — Настройка и Запуск

## ⚠️ Важно: Используйте Локальный Сервер

Сайт **ДОЛЖЕН** работать на локальном сервере, а не через `file://` протокол, иначе:
- ❌ Фото фильмов не загружаются
- ❌ Клики на фильмы не работают
- ❌ Авторизация может работать неправильно

## 🚀 Вариант 1: Python (Рекомендуется)

Если у вас установлен Python 3:

```bash
py server.py
```

Затем откройте браузер и перейдите на: **http://localhost:8001/**

Для остановки нажмите: **Ctrl+C**

## 🚀 Вариант 2: Node.js (Live Server)

Если у вас установлен Node.js:

```bash
npm install -g live-server
cd "e:\Сайт\Сайти кинотеатр куҳна а"
live-server
```

Браузер откроется автоматически на **http://localhost:8080/**

## 🚀 Вариант 3: VS Code Live Server

1. Установите расширение "Live Server" в VS Code
2. Кликните правой кнопкой на `index.html`
3. Выберите "Open with Live Server"

## ✅ Функции которые будут работать после запуска сервера

- ✓ Загрузка фото фильмов
- ✓ Нажатие на фильм → переход на страницу просмотра
- ✓ Поиск фильмов в реальном времени
- ✓ Добавление в избранное
- ✓ Авторизация и регистрация
- ✓ Персональное меню пользователя
- ✓ Мобильное меню (hamburger на телефоне)

## 📱 Тестирование функций

### Регистрация:
1. Нажмите кнопку "Вход"
2. Нажмите "Зарегистрируйтесь"
3. Введите:
   - Имя: любое имя
   - Номер: +998 XX XXX XX XX
   - Email: опционально
4. Нажмите "Зарегистрироваться"

### Вход:
1. Нажмите "Вход"
2. Введите номер телефона который вы использовали при регистрации
3. Нажмите "Войти"

### Просмотр фильта:
1. Нажмите на любой фильм в каталоге
2. Откроется страница с видео и информацией
3. Используйте кнопки "Добавить в избранное" и "Другие фильмы"

### Поиск:
1. Введите название фильма в поле поиска
2. Фильмы будут фильтроваться в реальном времени

## 🐛 Проблемы и решения

**Проблема**: "Фото не загружаются"
**Решение**: Убедитесь что открыли сайт на **http://localhost**, а не на `file://`

**Проблема**: "При нажатии на фильм ничего не происходит"
**Решение**: Снова - используйте локальный сервер!

**Проблема**: "Авторизация не работает"
**Решение**:
- Откройте DevTools (F12)
- Убедитесь что нет ошибок в консоли
- Очистите localStorage: `localStorage.clear()` в консоли
- Перезагрузите страницу

## 💾 Сохранение данных

Все данные сохраняются в **localStorage** браузера:
- `cinemaUsers` - данные пользователей
- `currentCinemaUser` - текущий залогиненный пользователь
- `cinemaFavorites` - избранные фильмы

При очистке кэша браузера все данные будут удалены!
