🔑 КЛЮЧЕВЫЕ JS FUNÇÃO ДЛЯ АВТОРИЗАЦИИ

- getCurrentUser()      - Получить текущего пользователя
- getAllUsers()         - Получить всех пользователей
- validatePhone()       - Проверить номер телефона
- normalizePhone()      - Привести номер к единому формату
- setCurrentUser()      - Установить текущего пользователя
- setupAuth()           - Инициализация системы авторизации
- handleLogin()         - Обработка входа
- handleRegister()      - Обработка регистрации
- handleLogout()        - Обработка выхода

💾 ХРАНИЛИЩЕ ДАННЫХ (localStorage)

Ключи:
- "cinemaUsers"        - Объект всех пользователей {phone: userData}
- "currentCinemaUser"  - Данные текущего пользователя
- "cinemaFavorites"    - Массив ID избранных фильмов

Формат пользователя:
{
  name: string,        // Имя пользователя
  phone: string,       // Номер в формате +998XXXXXXXXX
  email: string|null,  // Email (опционально)
  registeredAt: ISO8601, // Дата регистрации
  favorites: array     // ID избранных фильмов
}

🔧 КАСТОМИЗАЦИЯ:

1. Добавить свои фильмы:
   Отредактируйте массив `movies` в script.js
   Добавьте объект с полями: id, title, rating, image, video, description, genre, year, duration, featured

2. Изменить цвета:
   Отредактируйте CSS переменные в :root
   --accent: красный цвет
   --bg: цвет фона
   --text: цвет текста

3. Локальные видео:
   Замените URL видео на путь к файлам
   Например: "videos/my-movie.mp4"

4. Местные изображения:
   Замените URL изображений на пути к файлам
   Например: "images/poster.jpg"
