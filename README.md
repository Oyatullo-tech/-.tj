# 🎬 CineVerse — Онлайн кинотеатр

Современный, адаптивный и красивый онлайн-кинотеатр, построенный с использованием только **HTML, CSS и чистого JavaScript**.

## ✨ Особенности

- **Красивый Netflix-подобный дизайн** с темной темой и красными акцентами
- **100% адаптивный** — работает на телефонах, планшетах и ПК
- **Поиск в реальном времени** — быстрая фильтрация фильмов по названию
- **Система избранного** — сохранение любимых фильмов в localStorage
- **Видео плеер** — встроенный HTML5 плеер с полным управлением
- **Похожие рекомендации** — система подборки по жанрам
- **Анимации и переходы** — плавные эффекты при наведении и загрузке
- **Только ванильный JavaScript** — без библиотек и фреймворков

## 📁 Структура проекта

```
Сайти кинотеатр/
├── index.html          # Главная страница с каталогом фильмов
├── movie.html          # Страница просмотра фильма
├── style.css           # Все стили (темная классная тема)
├── script.js           # Весь функционал на JavaScript
└── README.md           # Этот файл
```

## 🚀 Быстрый старт

### Способ 1: Локальный сервер (рекомендуется)

```bash
# Перейти в папку проекта
cd "Сайти кинотеатр"

# Запустить Python HTTP сервер
python -m http.server 8000

# Открыть браузер
http://localhost:8000
```

### Способ 2: Прямое открытие

Просто откройте **index.html** в браузере двойным щелчком.

## 🎯 Функции

### 1. **Главная страница (index.html)**
- Hero блок с рекомендуемым фильмом
- Поиск по названию фильма
- Каталог фильмов в сетке (5, 3 или 1 столбец в зависимости от размера экрана)
- Раздел "Избранное" с вашими сохраненными фильмами
- Информационный раздел "Почему выбирают CineVerse"

### 2. **Страница фильма (movie.html)**
- Встроенный видео плеер HTML5
- Полная информация о фильме (название, год, жанр, рейтинг, описание)
- Кнопка добавления в избранное
- Рекомендации похожих фильмов
- Навигация обратно на главную

### 3. **Функции JavaScript**
- **Динамическая генерация карточек** из массива фильмов
- **Поиск в реальном времени** с фильтрацией
- **localStorage** для сохранения избранного между сеансами
- **URL параметры** для передачи ID фильма между страницами
- **Плавные анимации** при загрузке и наведении
- **Отзывчивое мобильное меню** для экранов < 768px

### 4. **CSS Эффекты**
- Темная тема с красными акцентами (#e50914)
- Задний фон с радиальными градиентами
- Эффекты при наведении с масштабированием и затемнением
- Skeleton loading для лучшего UX
- Адаптивные grid сетки (5/3/1 колонка)
- Плавный скролл с помощью `scroll-behavior: smooth`
- Кастомизированный скролл-бар

## 📊 Данные фильмов

Проект содержит 12 демо-фильмов из разных жанров:
- Sci-Fi: "The Silent Horizon", "Moonlit Code", "Orbit 9", "Neon Dreams"
- Action: "Crimson Night", "Velocity Run", "The Last Stand"
- Drama: "Echoes of Winter", "Last Frame", "Hearts in the Rain"
- Adventure: "Golden Tides"
- Thriller: "Midnight Protocol"

Каждый фильм содержит:
- ID и название
- Постер (картинка из Unsplash)
- Видео (демо видео из интернета)
- Рейтинг, год, жанр, длительность
- Описание

## 🔧 Требования

- Современный веб-браузер (Chrome, Firefox, Safari, Edge)
- JavaScript должен быть включён
- webP или JPEG для изображений
- MP4 или WebM для видео

## 💡 Советы по использованию

1. **Добавление своих фильмов**: Отредактируйте массив `movies` в `script.js`
2. **Изменение цветов**: Измените CSS переменные в `:root` в `style.css`
3. **Локальные видео**: Замените URL видео на пути к локальным файлам
4. **Оптимизация изображений**: Используйте сжатые изображения для быстрой загрузки
