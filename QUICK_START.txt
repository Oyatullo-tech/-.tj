# ⚡ Быстрый Старт

## Шаг 1: Запустите сервер (Python)

Откройте командную строку в папке проекта и напишите:

```
python server.py
```

Вы должны увидеть:
```
✓ Сервер запущен на http://localhost:8000/
✓ Откройте браузер и перейдите на http://localhost:8000/
✓ Для остановки нажмите Ctrl+C
```

## Шаг 2: Откройте браузер

Перейдите на: **http://localhost:8000/**

## ✓ Что должно работать:

✅ Фото фильмов загружаются
✅ Нажатие на фильм открывает его страницу
✅ Поиск фильмов работает
✅ Авторизация и регистрация работают
✅ Мобильное меню работает
✅ Добавление в избранное сохраняется

## 🆘 Если не работает

1. **Убедитесь что открыли `http://localhost:8000` а не `file://`**
2. Очистите кэш браузера (Ctrl+Shift+Delete)
3. Нажмите F5 чтобы перезагрузить страницу
4. Откройте DevTools (F12) и проверьте консоль на ошибки

Больше информации в файле **README_SETUP.md**
