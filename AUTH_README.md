# Система Авторизации CineVerse

## 📝 Описание

Реализована полная система регистрации и входа с номером телефона. Все данные пользователей сохраняются в `localStorage` браузера.

## 🚀 Функциональности

### 1. **Регистрация**
   - Введение имени пользователя
   - Введение номера телефона (+998 формат)
   - Email (опционально)
   - Валидация данных
   - Сохранение в localStorage

### 2. **Вход**
   - Вход по номеру телефона
   - Проверка наличия пользователя
   - Сохранение текущего пользователя

### 3. **Профиль**
   - Отображение телефона текущего пользователя
   - Кнопка "Выход"
   - Смена состояния UI при входе/выходе

### 4. **Хранение данных**
   - `cinemaUsers` - объект всех зарегистрированных пользователей
   - `currentCinemaUser` - текущий вошедший пользователь
   - Формат номера: `+998XXXXXXXXX`

## 📱 Форматы номеров телефонов (поддерживаемые)

Все эти форматы принимаются и нормализуются в `+9989XXXXXXXXX`:
- `+998 90 123 45 67`
- `+998 (90) 123 45 67`
- `+99890123456789` (12 цифр)
- `99890123456789` (без +)
- `89012345678` (старый формат приватизированного номера)

## 🔑 API Функций

### `validatePhone(phone)`
Валидирует номер телефона

**Параметр:**
- `phone` (string) - номер телефона для проверки

**Возвращает:**
- `boolean` - true если номер вален

### `normalizePhone(phone)`
Приводит номер к стандартному формату `+998XXXXXXXXX`

**Параметр:**
- `phone` (string) - номер в любом формате

**Возвращает:**
- `string` - нормализованный номер

### `getCurrentUser()`
Получает текущего вошедшего пользователя

**Возвращает:**
```js
{
  name: string,
  phone: string,
  email: string | null,
  registeredAt: ISO8601,
  favorites: array
}
```

### `getAllUsers()`
Получает объект всех пользователей (ключ = номер телефона)

**Возвращает:**
- `object` - все пользователи по номерам

### `setCurrentUser(user)`
Устанавливает текущего пользователя

**Параметр:**
- `user` (object | null) - пользователь или null для выхода

## 🎯 Использование в коде

```js
// Проверить, вошел ли пользователь
const user = getCurrentUser();
if (user) {
  console.log('Пользователь вошел:', user.name);
} else {
  console.log('Пользователь не вошел');
}

// Получить всех пользователей
const allUsers = getAllUsers();
console.log(Object.keys(allUsers).length, 'пользователей');

// Валидировать номер
if (validatePhone('+998901234567')) {
  console.log('Номер вален');
}

// Нормализировать номер
const normalized = normalizePhone('90 123 45 67');
console.log(normalized); // +998901234567
```

## 🎨 UI Элементы

### Header
- **Кнопка "Вход"** - если пользователь не вошел
- **Профиль пользователя** - если пользователь вошел
  - Отображает номер телефона
  - Кнопка "Выход"

### Модальное окно (#authModal)
- **Форма входа** (#loginForm)
- **Форма регистрации** (#registerForm)
- Переключение между формами

### Валидация
- Красные ошибки под каждым полем
- Валидация на отправку формы
- Отключение отправки при ошибках

## 🔒 Безопасность

- **Валидация на клиенте** - проверка формата
- **Нормализация номеров** - единообразное хранение
- **Нет паролей** - система на основе номера телефона
- **localStorage** - локальное хранилище (не облачное)

## ⚠️ Ограничения

- Данные хранятся только локально (в браузере)
- При очистке кэша браузера данные теряются
- Для продакшена требуется backend

## 🐛 Отладка

Откройте консоль браузера (F12) и используйте:

```js
// Увидеть всех пользователей
console.log(getAllUsers());

// Увидеть текущего пользователя
console.log(getCurrentUser());

// Очистить всех пользователей
localStorage.removeItem('cinemaUsers');
localStorage.removeItem('currentCinemaUser');
```
