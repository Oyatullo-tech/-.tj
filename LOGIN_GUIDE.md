# 🎬 Инструкция по использованию системы входа CineVerse

## 🚀 Быстрый старт

### Как зарегистрироваться:

1. Нажмите на кнопку **"Вход"** в правом углу header
2. В модальном окне нажмите **"Зарегистрируйтесь"**
3. Заполните форму:
   - **Ваше имя**: введите ваше имя (минимум 2 символа)
   - **Номер телефона**: +998 90 123 45 67 (или любой узбекский номер)
   - **Email**: (необязательно)
4. Нажмите **"Зарегистрироваться"**
5. После успешной регистрации вы автоматически войдете в аккаунт

### Как войти:

1. Нажмите на кнопку **"Вход"** в header
2. Введите номер телефона, на который вы регистрировались
3. Нажмите **"Войти"**
4. Готово! Вы в аккаунте

### Как выйти:

1. Посмотрите на header - там показан ваш номер телефона в профиле
2. Нажмите кнопку **"Выход"**
3. Вы вышли из аккаунта

## 📱 Поддерживаемые форматы номеров

Все эти форматы работают:

✅ `+998 90 123 45 67`  
✅ `+998 (90) 123 45 67`  
✅ `+99890123456789`  
✅ `99890123456789`  
✅ `09012345678`

## 🔐 Где хранятся данные?

- **Все пользователи**: в localStorage браузера (`cinemaUsers`)
- **Текущий пользователь**: в localStorage (`currentCinemaUser`)
- **Избранные фильмы**: в localStorage (`cinemaFavorites`) - сохраняются с профилем пользователя

⚠️ **Важно**: Если вы очистите данные браузера (кэш), все аккаунты будут удалены!

---

## 🛡️ Безопасность

- ✅ Валидация номера телефона
- ✅ Проверка на дублирование номеров
- ✅ Шифрование не требуется (сохранено локально)
- ❌ Пароли не используются (система на основе номера телефона)

## 🔧 Для разработчиков

Открыте консоль браузера (F12 → Console) и используйте:

```javascript
// Получить всех пользователей
console.table(getAllUsers());

// Получить текущего пользователя
console.table(getCurrentUser());

// Проверить, вошел ли пользователь
if (getCurrentUser()) {
  console.log('Пользователь вошел');
} else {
  console.log('Пользователь не вошел');
}

// Проверить валидацию номера
console.log(validatePhone('+998901234567')); // true или false

// Нормализировать номер
console.log(normalizePhone('90 123 45 67')); // +998901234567

// Очистить данные (для тестирования)
localStorage.removeItem('cinemaUsers');
localStorage.removeItem('currentCinemaUser');
console.log('Все данные очищены');
```

## 📋 API функций для разработки

### `getCurrentUser()`
Возвращает текущего вошедшего пользователя или `null`

```javascript
const user = getCurrentUser();
if (user) {
  console.log(`Привет, ${user.name}!`);
}
```

### `validatePhone(phone)`
Проверяет, валидный ли номер телефона

```javascript
if (validatePhone('+998901234567')) {
  console.log('Номер телефона правильный');
}
```

### `normalizePhone(phone)`
Приводит номер к единому формату

```javascript
const normalized = normalizePhone('90-123-45-67');
console.log(normalized); // +998901234567
```

### `getAllUsers()`
Возвращает объект всех пользователей

```javascript
const users = getAllUsers();
console.log(Object.keys(users).length); // количество пользователей
```

## ❌ Когда что-то не работает

**Проблема**: Кнопка входа не появляется  
**Решение**: Обновите страницу (F5)

**Проблема**: Форма говорит, что пользователь не найден  
**Решение**: Может быть, вы используете другой формат номера или браузер. Попробуйте зарегистрироваться заново.

**Проблема**: Потеряны все аккаунты  
**Решение**: Возможно, вы очистили кэш браузера. К сожалению, данные хранятся локально.

**Проблема**: Ошибка валидации номера  
**Решение**: Убедитесь, что используете узбекский номер (+998)
